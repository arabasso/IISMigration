# IISMigration
## IIS migration tool

```
Usage: IisMigration.exe [OPTION]... [FILE]...
Export/Import IIS Configurations.

  /export [FILE.XML]             Export Pools and Sites.
  /export-pools [FILE.XML]       Export Pools.
  /export-sites [FILE.XML]       Export Sites.

  /import [FILE.XML]             Import configurations
  /import-users [FILE.XML]       Import users to Active Directory
  /import-dirs [FILE.XML]        Create directory structure from IIS configuration
```
